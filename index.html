Manipulação de Strings em PHP

Em PHP, há 4 formatos de strings, sendo eles Double quoted,Single quoted, Heredoc, Nowdoc. Deve-se lembrar que em PHP, existe uma diferença entre stringo de aspas duplas (Double quoted) e aspas simples (Single quoted)

A diferença entre a Double Quoted e a Single Quoted é a interpretação que o PHP fará, ou seja, existe uma interpretação de conteúdo

Pra concatenarmos uma string com outra, utilizamos "." ao invés de "+", como nas outras linguagens.

Obtendo Dados de Formulários

No PHP, quem estará processando os dados, será o servidor. Por isso, diferente de JS, há algumas infromações que o formulário não poderá ser esquecido. Dentre essas infromações, as que não poderá ser esquecida são os métodos (method) e as ações(action)

Os method nada mais são do que o método em que os dados serão codificados para serem enviados ao servidor. Por exemplo, o método get ou post

Já os action será para onde serão enviados os dados do formulário. Por exemplo, um arquivo formulario.php
